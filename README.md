## ≤░Υ╛ WPM Calculation

typeflux uses the standard WPM formula:
- **Gross WPM** = (Characters Typed / 5) / Minutes
- **Net WPM** = Gross WPM - (Errors / Minutes)
- 5 characters = 1 standard word
